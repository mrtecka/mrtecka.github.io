Cheatsheet Table

Item Name Type Description
<h1> Header Tag The wrapper for the Subject or Main Text
<table> Table Tag Used to create a table
class Class Attribute Used to name a tag or section. Can be reused throughout the html file.
id ID Attribute Used to name a tag or section. Can NOT be reused in the html file.
<img> Image Tag The wrapper for adding images to your file.
